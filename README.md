# Full Stack Blog Application

## 📌 Project Description
This is a full stack blog application developed as part of my final year project. It allows users to create, read, update, and delete blog posts.

## 🚀 Technologies Used
- Frontend: Next.js
- Backend: Next.js API Routes
- Styling: Tailwind CSS
- Database: MongoDB

## ✨ Features
- User can create blog posts
- Edit and delete posts
- Responsive design
- Fast performance using Next.js

## 🎯 Purpose
This project was built to understand full stack development and how frontend, backend, and database work together.
